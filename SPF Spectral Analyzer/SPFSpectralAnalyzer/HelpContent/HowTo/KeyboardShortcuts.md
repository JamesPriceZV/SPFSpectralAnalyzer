## Overview
Use these shortcuts to move quickly through common actions.

## Global
- Settings: Command + ,
- Undo: Command + Z
- Redo: Shift + Command + Z

## Dock Actions
- Import: Command + I
- Run Pipeline: Shift + Command + R
- AI Analysis: Shift + Command + A
- Export: Command + E
