SPF Spectral Analyzer — Application Workflow

Architecture map generated from source code · SPFSpectralAnalyzer · March 2026

1 · App Launch
🚀SPFSpectralAnalyzerApp (@main)
  • Initialises DataStoreController (SwiftData + CloudKit)
  • Initialises InstrumentManager (BLE / mock driver)
  • Starts ConsoleCapture (diagnostic logging)
  • Renders app icon at runtime (macOS)
  • Registers macOS menu commands: Diagnostics, Help, Export HTML, Instrument Control
🗄️DataStoreController
  • Creates SwiftData ModelContainer
  • Attempts CloudKit sync; falls back to local SQLite on failure
  • Stores StoredDataset + StoredInstrument models
  • Records store-reset history in UserDefaults
🪟Windows (macOS)
  • Main window → RootContentView → ContentView
  • Diagnostics Console → DiagnosticsConsoleView
  • Help → HelpView
  • Instrument Control → InstrumentControlView
  • Settings → SettingsView (Settings scene)
Injects environment objects
2 · Main View — AppMode Tab Navigation
📁Data Management Tab
  • Browse & search stored datasets
  • Import SPC / CSV files (file picker)
  • Assign role: Reference (known SPF) or Prototype
  • Assign substrate plate type, application quantity, instrument
  • Archive / permanent-delete datasets
  • Duplicate cleanup tool
  • iCloud sync status bar
📈Analysis Tab
  • Spectrum chart (Swift Charts, 280–420 nm)
  • Sidebar: spectrum list, filtering, sorting
  • Processing controls (alignment, smoothing, baseline, normalisation)
  • SPF display panel (COLIPA / ISO 24443 / HDRS)
  • Dashboard metrics card
  • AI Analysis section
  • Peak detection overlay
📄Reporting Tab
  • Export: CSV, Excel (.xlsx), PDF, HTML report
  • Options: title, operator, notes, processing info, metadata
  • Chart snapshot sharing
  • Schedule calendar event (CalendarService)
⚙️Settings
  • SPF calc method & factors
  • AI provider & API keys
  • iCloud sync toggle
  • UI preferences
📷Camera (iOS)
  • SpectralCameraView
  • VisionColorAnalyzer
  • Photo picker
User imports files
3 · Data Ingestion
📂File Import (PlatformFilePicker)
  • Accepts .spc (Shimadzu binary), .csv, .txt
  • Multi-file drag-and-drop or Open panel
  • Append or replace existing spectra
🔬SpectrumParsingWorker
  • ShimadzuSPCParser — binary SPC/GSPC header parsing
  • CompoundFile — OLE2 compound document reader
  • CSV/text parser (wavelength + absorbance columns)
  • FilenameMetadataParser — extracts SPF label, date, batch from filename
  • Validation against known wavelength range & point count
🏷️SPFLabelStore / SPFLabelMap
  • Fuzzy-matches filename to known commercial SPF labels
  • Provides reference SPF value for calibration
Parsed spectra → StoredDataset (SwiftData)
4 · Persistence & Sync
💾StoredDataset (@Model)
  • Binary spectrum data (x/y via SpectrumBinaryCodec)
  • File hash for duplicate detection
  • Role (reference / prototype), knownInVivoSPF
  • Plate type, application quantity (mg), formulation type
  • HDRS tags JSON, instrument ID
  • Archive flag + timestamp
🎛️StoredInstrument (@Model)
  • Name, serial, model, lab
  • Calibration date, wavelength range
  • InstrumentCatalog — predefined Shimadzu RF-series entries
☁️CloudKit + iCloud Sync
  • NSPersistentCloudKitContainer
  • CloudKitSyncMonitor — tracks push/pull events
  • ICloudSyncCoordinator — UserDefaults sync status keys
  • CloudSyncProgressView — inline status HUD
  • Graceful fallback to local store when CloudKit unavailable
🗂️DatasetPersistenceService
  • Save new datasets to ModelContext
  • Session restore (last-opened IDs in UserDefaults)
  • Duplicate detection by file hash
  • Archive & permanent delete helpers
Spectra loaded into AnalysisViewModel

6 · Metrics & SPF Computation (async, background actor)
🧮SpectralMetricsWorker
  • Runs on background actor
  • SpectralMetricsCalculator: CriticalWavelength, UVA/UVB ratio, COLIPA SPF integral
  • Builds calibration curve from reference datasets
  • Finds nearest reference match
  • Computes dashboard metrics (averages across all spectra)
☀️SPF Estimation
  • SPFEstimationResolver — picks best SPF estimate
  • Tiers: Calibrated → COLIPA Raw → CoreML → C-Factor override
  • c-factor, substrate correction, adjustment factor applied
  • Method: COLIPA (ISO 24443) or ISO 23675 (HDRS)
🎯Calibration Engine
  • Linear regression: spectral metrics → in-vivo SPF
  • Reference datasets (known SPF) feed the model
  • Reports R², RMSE
  • CIEErythemalSpectrum & ColipaSpectralData standard reference data
📐HDRS / ISO 23675
  • HDRSCalculator — HDRS ratio per sample group
  • Auto-tag spectra by plate type & irradiation state
  • Product type: emulsion / stick / powder
  • Results stored in hdrsResults[ ]
🤖CoreML Prediction
  • SPFPredictionService — on-device ML model
  • SPFModelSchema — feature schema (absorbance at key wavelengths)
  • MLTrainingService — trains from reference datasets
  • Prediction cached in coreMLPrediction
Results update cached* properties → UI re-renders
7 · AI Analysis
🧠AIViewModel
  • Holds AI runtime state: result, structuredOutput, history
  • Token estimation, cost tracking
  • History of up to 20 entries (AIHistoryEntry)
  • Sidebar structured sections: insights, risks, actions
🔀AIProviderManager
  • Auto mode: on-device first → OpenAI fallback
  • FoundationModelsProvider — Apple Foundation Models (on-device)
  • OpenAIProvider — GPT via configurable endpoint & model
  • API key stored in KeychainStore
📊SpectralMetricsTool
  • Tool-use interface given to the AI model
  • Exposes: SPF value, critical wavelength, UVA/UVB ratio, calibration R²
  • Prompt presets: Summary, Regulatory, Comparative, Custom
  • Structured JSON output: AIStructuredOutput
💬AI Output & Saving
  • Displayed in AISection / AIAnalysis panel
  • User prompted to save: Save to File or Open
  • Result saved as .txt with timestamp
  • History browsable in the sidebar
User triggers export
8 · Export & Reporting
📊CSV Export
  • Wavelength + all spectra columns
  • Optional metadata header comments
  • Processing settings embedded
  • SPF metrics (COLIPA, estimated, calibrated)
📗Excel (.xlsx) Export
  • OOXMLWriter — direct OOXML generation (no dependencies)
  • Spectral data + metrics sheet
  • Per-spectrum summary rows
📕PDF Report
  • PDFReportRenderer — CoreGraphics / PDFKit renderer
  • PDFReportModels — typed section model
  • Chart image embedded via ChartSnapshotRenderer
  • Operator name, title, notes, metrics table
🌐HTML Report
  • HTMLReportWriter — self-contained HTML + inline chart
  • Datasets snapshotted before file-save dialog (CloudKit safety)
  • Menu command: Export HTML Report (macOS)
  • Timestamp auto-appended to filename
🔗Share & Calendar
  • ShareService — UIActivityViewController / NSSharingService
  • ShareSheet — SwiftUI sheet wrapper
  • CalendarService — adds EKEvent with report details
  • ScheduleEventSheet — user picks date/type
Cross-Cutting Concerns
🔍Search Engine
  • SearchQuery — boolean AND/OR/NOT parser
  • SpectrumSearchRecord, DatasetSearchRecord
  • Sidebar real-time filtering
  • SearchSyntaxHelpView — in-app help popover
✅Validation
  • ValidationService — wavelength range, point count, NaN/Inf
  • InvalidSpectrumItem — quarantined bad spectra
  • ValidationLogEntry — timestamped audit trail
  • SpectrumValidation — import-time pre-check
📋Diagnostics & Logging
  • Instrumentation — structured log (area + level + duration)
  • ConsoleCapture — tees stdout/stderr to in-memory buffer
  • DiagnosticsConsoleView — floating log window (macOS)
  • DiagnosticsLogModels — typed log entry model
🎛️Instrument Control
  • InstrumentManager — BLE / mock driver abstraction
  • InstrumentCatalog — predefined RF-series models
  • InstrumentControlView — live measurement panel
  • AssignInstrumentSheet — link instrument to dataset
📱Platform Abstraction
  • PlatformFilePicker / PlatformFileSaver
  • PlatformPasteboard (macOS / iOS)
  • PlatformColors (adaptive tints)

  • GlassModifiers — Liquid Glass effect helpers
